# E-Commerce Sales Dashboard (Power BI)

This project is an **interactive Power BI dashboard** built to analyze and visualize e-commerce sales performance using real-world transactional data.

The dashboard provides insights into **sales amount, profit, quantity, customer behavior, payment modes, and regional performance** with interactive slicers for deeper analysis.
